# Duck Jam

Winner — Duck Sauce Game Jam ([ducksauce.games](https://www.ducksauce.games/))

An autoscrolling river game where collecting ducks layers a live melody. Each duck you pick up adds a new layer to the music, building the track as you go.

Read more about the project at [ruaridhmurdoch.com](https://ruaridhmurdoch.com).

## About this repo

This repository is for team collaboration during the jam — it is **not set up to be run or played**. Feel free to browse the code.

Built with Unity, all scripts in C#.
